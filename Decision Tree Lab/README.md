# Decision Tree EMV Studio

Open `index.html` in a browser to use the tool.

Students can:

- Build decision node, probabilistic node, and end branches.
- Define branch counts before a node is drawn.
- Enter short branch labels, probabilities, and branch cash flows.
- Calculate full-path terminal payoffs as a separate step.
- Roll the tree back using expected monetary value after payoffs are calculated.
- Highlight the best decision branch.
- Drag nodes to arrange the tree manually.
- Step through plain-language rollback explanations.
- Save locally or copy/load the tree as JSON.

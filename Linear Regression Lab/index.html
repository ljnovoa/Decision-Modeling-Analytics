Linear Regression Lab

A FIFA-inspired simulation of how sample regression lines estimate a hidden population relationship.

Designed by Luis Novoa and built using Codex (inspired by class video by Dr. Scott Stevens - for educational purposes)

The Analytics Brief

  • FIFA Analytics researchers developed the Attacking Threat Index (ATI) to measure attacking danger during a 90-minute match.
  • ATI combines dangerous attacks, penalty-area entries, shots on target, and successful through passes.
  • ATI has no fixed limits, although it is usually between 0 and 100.
  • The research question is whether a team's Possession Time (PT), measured in minutes, is related to ATI.
  • For this pedagogical lab, the true population relationship is known. Random samples show how sample regression lines estimate that hidden population line.

    To generate the population data, the following assumptions were made:

    • Population relationship: ATI = 10 + 0.90(PT) + ε
    • Random error: ε ~ N(0, 122)
    • Possession Time distribution: PT ~ Uniform(15, 75)
    • Generate 150 PT values and 50 ATI values for each PT value.
    • Total population size: 7,500 observations.
